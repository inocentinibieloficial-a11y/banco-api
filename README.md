# 💰 Banco API - Conversor de Moedas

Projeto desenvolvido em **Java + Spring Boot** com integração a API externa de cotações em tempo real.

## 🚀 Funcionalidades

✔ Consulta de moedas em tempo real
✔ Conversão de Real (BRL) para outras moedas
✔ Menu interativo no terminal
✔ Consumo de API REST
✔ Tratamento de erros
✔ Código organizado em camadas

## 💻 Tecnologias utilizadas

* Java
* Spring Boot
* Maven
* Jackson
* HTTP Client
* API REST

## 🌎 Moedas disponíveis

* USD = Dólar Americano
* EUR = Euro
* ARS = Peso Argentino
* CLP = Peso Chileno
* UYU = Peso Uruguaio

## ▶ Como executar

```bash
mvn spring-boot:run
```

Ou execute a classe principal no IntelliJ.
